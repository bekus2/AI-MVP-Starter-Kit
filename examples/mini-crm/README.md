# Mini CRM Example

This example is a small browser-only CRM demo.

## How To Run

Open:

```text
examples/mini-crm/index.html
```

## Features

- Add a lead.
- List saved leads.
- Update lead status.
- Delete a lead.
- Persist demo data in LocalStorage.
- Escape rendered user-entered values.

## Security Note

This is a local demo. A production CRM needs backend authentication, server-side validation, authorization, audit logging, backups, and privacy controls.
