# Lead Form Example

This example demonstrates a static lead form using LocalStorage.

## How To Run

Open:

```text
examples/lead-form/index.html
```

## What It Demonstrates

- Client-side validation.
- LocalStorage persistence for demo data.
- Safe HTML escaping before rendering user-entered values.
- Clear status messages.

## Security Note

This example is not a production lead capture backend. Real lead collection should use server-side validation, spam protection, secure transport, and privacy-aware storage.
