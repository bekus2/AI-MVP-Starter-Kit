# Landing Page Example

This example is a simple static landing page for validating one MVP message.

## How To Run

Open:

```text
examples/landing-page/index.html
```

## What It Demonstrates

- Semantic static HTML.
- Responsive layout.
- Single value proposition.
- Small feature grid.
- No dependencies or external assets.

## How To Extend

- Add a real lead form.
- Add product screenshots.
- Add a pricing or FAQ section.
- Connect analytics only after privacy and compliance review.
