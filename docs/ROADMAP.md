# Roadmap

This roadmap is practical and intentionally conservative.

## v0.1.0 — Initial Static Toolkit

- Multilingual static app.
- MVP brief generator.
- AI task generator.
- Readiness checklist.
- Demo lead form.
- AI documentation files.
- Starter template.
- Runnable examples.
- Validation script.

## v0.2.0 — More Starter Templates

- Portfolio starter.
- SaaS dashboard starter.
- Admin panel starter.
- Service business starter.

## v0.3.0 — Export Generated Prompts As Markdown

- Export brief as `.md`.
- Export AI task as `.md`.
- Include timestamp and selected language.

## v0.4.0 — GitHub Issue Templates

- Bug report template.
- Feature request template.
- Security review checklist issue.
- AI task issue template.

## v0.5.0 — Optional Backend Starter

- Minimal backend starter.
- Environment configuration.
- Secure form handling.
- Example persistence layer.

## v1.0.0 — Stable Release

- Stable file structure.
- Expanded docs.
- Stronger validation.
- Browser-level test coverage.
- Polished examples and contribution workflow.
