# Changelog

All notable changes to AI MVP Starter Kit are documented here.

## 2026-06-16 — v0.1.0

### Summary

Initial public release of the static AI MVP Starter Kit.

### Added

- Multilingual static web app with English, Russian, and Kazakh UI support.
- MVP Project Brief Generator.
- AI Agent Task Generator.
- MVP Readiness Checklist with LocalStorage persistence.
- Starter Project Structure Viewer.
- Local demo lead form with safe rendering.
- AI-agent documentation files in `ai/`.
- Practical documentation in `docs/`.
- Reusable starter template in `starter/`.
- Runnable examples in `examples/`.
- Repository validation script in `scripts/validate.mjs`.

### Security

- Added frontend HTML escaping helper.
- Documented LocalStorage limitations.
- Added `.env.example` without real secrets.
- Added `SECURITY.md` with reporting and secure development guidance.

### Notes

- This release is intentionally dependency-free and does not include backend code or deployment automation.
